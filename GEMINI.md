# Netlify Free Target Rules
- Framework: Next.js (Static Export mode - output: 'export'). Pure HTML/CSS/JS only.
- Language: TypeScript, Node.js environment. Do NOT use Python, PHP, or Java backend scripts.
- Forms: Use standard HTML forms containing the 'data-netlify="true"' attribute. 
- Aesthetic: Award-winning premium dark mode, Tailwind CSS, Framer Motion animations.
